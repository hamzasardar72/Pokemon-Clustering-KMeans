# Centroid-Based Clustering on the Pokemon Dataset

## Overview
Applied K-Means clustering on 800 Pokemon based on battle statistics 
(HP, Attack, Defense, Sp. Atk, Sp. Def, Speed).

## Key Results
- Optimal K = 5 (Elbow + Silhouette method)
- Legendary Pokemon mostly grouped in high-stat cluster
- PCA used for 2D/3D visualization

## Technologies Used
- Python, Pandas, NumPy
- Scikit-learn (KMeans, PCA, StandardScaler)
- Matplotlib, Seaborn

## Files
- `Pokemon_Clustering.ipynb` — Main notebook
- `pokemon.csv` — Dataset
